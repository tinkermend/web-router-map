# 网站菜单路由地图构建器

### 2.2 技术栈选型

| 模块         | 技术选型              | 说明                   |
| ------------ | --------------------- | ---------------------- |
| Web 框架     | FastAPI               | 高性能异步 API 框架    |
| 浏览器自动化 | Playwright            | 现代浏览器仿真引擎     |
| 任务调度     | APScheduler           | 灵活的 Python 调度框架 |
| 数据验证     | Pydantic              | 数据模型验证           |
| 状态管理     | Redis                 | 分布式锁与缓存         |
| 数据存储     | PostgreSQL + pgvector | 关系型存储 + 向量检索  |
| 加密         | Cryptography          | 敏感信息加密           |
| 日志         | Loguru                | 结构化日志管理         |
| 监控         | Sentry                | 异常监控与报警         |
| 验证码       | dddddocr              | 验证码识别             |
